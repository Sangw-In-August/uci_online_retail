프로젝트 개요

목표

  • 스토어 운영 최적화
    • 재고 관리를 포함한 효율적인 운영 방안 모색
    • 구매 트렌드 분석을 통한 운영 전략 개선
    • 이상치(Outlier)를 더 잘 이해하고 대응
  • 고객 만족도 개선
    • 반품된 상품에 대해 심층 분석하여 개선 방안 도출
    • 고객 경험 향상을 위한 데이터 기반 접근

데이터 세트 개요

: 약 541,000건의 온라인 스토어에서 발생된 거래 데이터

  • InvoiceNo ㅡ 여러 품목이 함께 구매될 때 사용되는 고유 식별자
  • StockCode ㅡ 제품을 나타내는 고유 식별자
  • Description ㅡ 제품에 대한 설명
  • Quantity ㅡ 구매된 제품의 수량
  • InvoiceDate ㅡ 구매 시 발행된 송장 날짜 및 시간
  • UnitPrice ㅡ 제품 1개당 가격
  • CustomerID ㅡ 고객을 나타내는 고유 식별자
  • Country ㅡ 제품이 구매된 국가

1) 반품 거래 스케일 확인

returns_filtered = returns[~returns['Description'].isin(filter_outs)].reset_index(drop=True)

3) 자주 구매되는 품목 비교

: 회원과 비회원간 자주 구매하는 품목을 비교

  • (코드 보기)
    # 자주 구매되는 품목 비교
# 1) 비회원 분석
# - 품목별 판매량
unknown_quantity = customer_unknown_df.groupby('Description').size()

# - 품목별 판매 비율
unknown_quantity_normalized = unknown_quantity / unknown_quantity.sum() * 100

# 2) 회원 분석
# - 품목별 판매량
known_quantity = customer_known_df.groupby('Description').size()

# - 품목별 판매 비율
known_quantity_normalized = known_quantity / known_quantity.sum() * 100

# 3) 품목별 차이
# - 비교를 위한 데이터 세트
comparison_df = pd.DataFrame({
    '비회원 구매 비율': unknown_quantity_normalized,
    '회원 구매 비율': known_quantity_normalized
})

# - 둘 간의 차이 산출
comparison_df['차이'] = comparison_df['비회원 구매 비율'] - comparison_df['회원 구매 비율']

# - 차이 Top/Bottom 상위 5개 추출
top5 = comparison_df.sort_values(by='차이').head(5)
bottom5 = comparison_df.sort_values(by='차이', ascending=False).head(5)

print('회원 vs 비회원간 자주 구매되는 품목 비교: \n')
print(pd.concat([top5, bottom5]))

분석 방법

  • 회원과 비회원 데이터 세트에 대하여 품목별 구매된 빈도가 얼마나 많은지를 비교
  • 표준화 처리하여 차이 계산

차이 크기 비교

  • 둘 간의 차이가 (-) 값이 클 수록 회원에 의해서만 구매된 품목임
  • 둘 간의 차이가 (+) 값이 클 수록 비회원에 의해서만 구매품목임

비회원에 의해 주로 구매된 품목

→ 이에 대한 심층 분석 필요를 느낌

전체 품목별 거래액 중 DOTCOM POSTAGE가 가장 높은 거래액을 기록

4) 반품 비율 차이

: 회원, 비회원 여부별 전체 거래액 중 반품된 거래액 비율 차이 확인

  • (코드 보기)
    # 회원/비회원 구분
data_filtered['CustomerID_yn'] = np.where(data_filtered['CustomerID'].isna(), '비회원', '회원')

# 반품된 거래액 계산 (반품 거래는 음수이므로 절댓값 변환)
data_filtered['return_sales'] = np.where(data_filtered['Sales'] < 0, -data_filtered['Sales'], 0)

# 회원 여부별 전체 거래액 (반품 포함)
total_sales_by_customerGroup = data_filtered.groupby('CustomerID_yn')['Sales'].sum()

# 회원 여부별 반품된 거래액 합계
return_sales_by_customerGroup = data_filtered.groupby('CustomerID_yn')['return_sales'].sum()

# 반품율 계산 (반품 거래액 / 전체 거래액)
return_ratio_comparison = return_sales_by_customerGroup.divide(total_sales_by_customerGroup)

# 결과 출력
print('회원 여부별 반품율 비교: ')
print(return_ratio_comparison)
  • 출력 결과
    회원 여부별 반품율: 
- 비회원   0.0030
- 회원    0.0548

⇒ 회원과 비회원 간 반품 비율에 뚜렷한 차이가 나타남. 비회원 주문의 반품 비율은 전체 거래액의 약 0.3%에 불과한 반면, 회원 주문의 반품 비율은 약 5.5%로 상대적으로 높은 수준을 보임

인기 제품 분석

: 인기 제품(매출액 기준 상위 10개 제품)에 대한 구매 지역별, 고객별 분석 수행

1) 구매 지역별 분석

  • 앞서 살펴보았듯이 전체 거래의 91%는 영국에서 발생되었음
  • 영국에서 대부분 판매되는 제품(Domestic) vs 해외 판매 비중이 높은 제품으로 구분 가능성 확인
    • 일부 제품은 해외 매출 비중이 매우 높음

2) 고객별(CustomerID) 분석

  • 파란색으로 표시된 부분이 Customer ID가 식별되지 않은 비회원
  • 일부 제품의 경우 특정 고객에 의해 대량으로 구매되었음을 알 수 있음
    • 도매상에 의해서 대량으로 구매되었을 것으로 추측

인기제품 시계열 분석

: 국내(영국)과 해외의 시간의 따른 인기 제품(상위 매출액 10위 제품) 의 변화를 추적

국내(영국) 인기 제품 변화

해외 인기 제품 변화

도출 가능한 주요 인사이트

  • 시기별로 인기 제품의 주요 변화를 확인
  • RABBIT NIGHT LIGHT 제품은 2011년 5월 부터 인기 제품 리스트에 올랐으며 해외에서 먼저 반응을 보이다가 매출 증가 확인
    • 이후 국내(영국)에서 2011년 11월에 매출이 큰 폭으로 증가
  • 앞서 살펴보았듯이 POSTAGE 제품은 국내 보다는 해외에서 꾸준한 인기를 얻고있음

반품 비율 추이

: 시기별로 전체 거래액 대비 반품된 거래액 비율을 추적

  • 전체 기간 동안 평균 반품율은 약 3.8% 이나 시기에 따라 반품율이 치솟는 시점이 존재
  • 특정 시기에 반품율이 높다는 것은 비즈니스 상에 어떤 문제가 발생했다는 시그널로 이에 대한 추가분석이 필요할 것으로 보임
    • 하지만 이에 대한 심화 분석은 추후 분석 과제로 남겨둘 예정

비즈니스 제언

반품된 거래

  • 일부 제품은 반품율이 매우 높게 나타나는데, 이에 대한 추가적인 분석을 통해 원인을 파악하고 시정할 필요가 있음
  • 또한, 반품율이 치솟는 시점이 존재하는데 이를 해결해야 할 ‘문제’라고 인식하고 원인을 식별하고 대처 방안 마련

매출 양상

  • 국내 매출과 해외 매출의 양상은 일부 차이를 보임
  • 지역별 인기 제품, 성수기(매출 증가 시기)에 대하여 이해하고, 이에 대한 Supply Chain에 대한 점검 필요

고객

  • 일부 고객은 대량 구매를 기반으로 한 도매업자일 가능성이 높으며, 이들은 장기적인 거래 유지가 중요한 핵심 고객이 될 수 있음.
    • 이러한 고객을 대상으로 대량 구매 시 추가 할인 정책을 도입하는 방안을 고려할 필요가 있음
